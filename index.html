Instructions

This map stores and visualizes ECC Site Review Data for project teams. To generate a 5 and 10 minute walkshed, hold control key and click on map location - the area that one is able to walk to in 5-10 minutes is bound by blue polygons. Clicking on map returns the data specified in menu. Using the draw features tool to draw and label features on map. Use Export button to share created data.
